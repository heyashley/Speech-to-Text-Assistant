# 🎤 AI Speech to Text

An AI-powered web application that converts speech into text using the Whisper model.  
Users can either **speak through the microphone for live transcription** or **upload an audio file** to convert speech into text.

---

## 🚀 Features

- 🎙 Live speech-to-text using microphone
- 📂 Upload audio files for transcription
- 🧠 AI-based speech recognition using Whisper
- 🌊 Waveform animation while speaking
- 🎨 Modern gradient UI with animated microphone
- ⚡ Real-time transcript display

---

## 🛠 Tech Stack

Frontend:
- HTML
- CSS

Backend:

AI Model:
- Whisper (OpenAI)
